# Recibos y soportes

Agrega aquí los recibos, facturas, comprobantes de transferencia y demás soportes de las compras registradas.

Convención recomendada para los nombres:

`AAAA-MM-DD_proveedor_concepto_valor.ext`

Ejemplo:

`2026-07-12_ricaurte_stickers_45000.pdf`

Cada gasto del archivo **Registro de inversiones.xlsx** debe poder relacionarse con un soporte de esta carpeta.
